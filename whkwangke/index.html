网课推荐（知乎答案收集）

发表于 2020-07-12 | 分类于 其他 ， 文化课 | | 本文总阅读量 次
字数统计: 1.2k | 阅读时长 ≈ 4

这里是jwj翻了很多知乎答案收集得到的结果，希望对大家有所帮助qwq

语文

  • 乘风-腾讯课堂x9
    • 提分快，适合理科生
    • 文言文
  • 杨洋-腾讯课堂x8
    • 培养思维，作文讲的很好
  • 国家玮-新东方、大黄蜂课堂x6
    • 社科文古文都讲的不错
    • 偏向于思维提升，讲法别出心裁
  • 董腾-有道精品课x2
    • 幽默风趣应试很强
  • 王大绩-简单学习网
  • 谢明波
  • 王丹宁-原新东方
  • 陈焕文-乐学
    • 理科思维，理解难度会大

数学

  • 周帅-高德x9
    • 每一个知识点都讲得特别全面，各个题型都有涉及
    • 偏基础一点
    • 授课内容集中、信息量大，讲题侧重培养思维，而非机械操作，课时多，知识点细。
  • 郭化楠-有道精品课、学而思x5
    • 重视基础
    • 非常注重基础，潜移默化的拓展知识之间的延伸
    • 拔高（有道精品课）
    • 全网最完美的“自学”视频，适合零基础。把知识点的来龙去脉讲的透彻，可以培养出真正的学术性人才
  • 赵礼显-跟谁学x5
    • 拔高
    • 课时少，课程主讲题型解题法，对基础知识讲解少，不适合基础差的人群。短期提分快。
  • 凉学长-腾讯课堂x3
    • 方法巧妙
  • 蔡德锦-腾讯课堂x2
    • 通过带你做题来深刻的理解这个知识点，适合拔高的同学
    • 给人踏实很值得信赖的感觉，讲的真的很吸引人，效率很高，方法总结很到位，简介明了
  • 鲲哥带你学数学-哔哩哔哩x2
  • 蒋叶光-领世培优
  • 建哥数学
  • 高中数学北师math杨-哔哩哔哩
  • 佟大大是ETT-哔哩哔哩
  • 讲高中数学的邓老师-哔哩哔哩
  • 高中数学蔡德锦老师-哔哩哔哩
  • 猪猪老师教数学-哔哩哔哩
  • 成都高中数学cdgzsxfml-哔哩哔哩
  • 司马红丽老师-精华在线
  • 李永乐老师-原精华、美提课堂
    • 思维训练
  • 王梦抒-腾讯课堂
  • 数学超人-腾讯课堂
  • 关旭-前有道
    • 圆锥曲线
  • 蒋叶光
    • 强推他的圆锥曲线，全网算是比较好的了,其他课程拔高，门槛较高
  • 王文勇老师-勇哥数学APP

英语

  • 徐磊-跟谁学x10
    • 磊神，名不虚传
  • 李辉-有道精品课x4
    • 方法论好，提分快
  • 刘晓峰老师-原新东方
  • 麻雪玲老师-简单学习网

物理

  • 坤哥-腾讯课堂x7
    • 构造一个完美的物理知识体系，题型全部在他的课程之中，非常的优秀
    • 基础
  • 王羽-腾讯课堂x6
    • 拔高
  • 刘杰-有道精品课x3
    • 拔高
  • 坤哥物理-哔哩哔哩x2

    • 拔高
  • 夏梦迪老师-新东方x2
    • 他能从最基础的内容说到最深层的知识，循序渐进，课堂生动有趣，有基础也有大招，同样课时较多
    • 拔高
  • 陈老师敲黑板-哔哩哔哩
    • 清华附中物理老师，力学电学内容都有
  • 物理X学姐-哔哩哔哩
    • 南京大学天文系的一位学姐，颜值与才华并存的一位up主，非常优秀。
  • 大宽物理-哔哩哔哩
  • 曾珍
  • 珍哥物理
  • 郑少龙老师-猿辅导

化学

  • 陆艳华-伴学优课、金榜在线x8

    • 选五看陆姐
    • 全网风评最好的老师，没有之一
  • 李政-乐学x8
    • 适合基础好的，完全不一样的眼界和视角，电化学直接打开新世界的大门
    • 选三看政神
  • 木子-腾讯课堂x2
    • 通俗易懂，方法好
  • 高东辉老师-精华在线
    • 基础
