    导数¶

    基础¶

    \forall x,a>f(x)\iff a>f(x)_{max}
    \forall x,a<f(x)\iff a<f(x)_{min}

    极限¶

    数列极限：当 n→\infty 时，a_n 是否趋近于一个常数？方法：分式上下同时除以 n 或 n 的多次方以使上下都可导。高阶除低阶无极限。

    函数极限：若 f(x) 图像在点 x=x_0 处连续且 f(x_0) 存在，则 {\lim_{x→x_0}f(x)=f(x_0)} 。

    求导运算¶

    平均变化率¶

    {f(x_2)-f(x_1)\over x_2-x_1}={Δy\over Δx}

    定义式¶

    这里为了帮助理解导数的几何意义没有去掉 x_0 。

    f'(x_0)=\lim_{Δx→0}{Δy\over Δx}=\lim_{Δx→0}{f(x_0+Δx)-f(x_0)\over x_0+Δx-x_0}

    基本公式¶

    f(x)=c\implies f'(x)=0，c 为常数
    f(x)=x^\alpha\implies f'(x)=\alpha x^{\alpha-1}，\alpha \in Q^*
    f(x)=\sin{x}\implies f'(x)=\cos{x}
    f(x)=\cos{x}\implies f'(x)=-\sin{x}
    f(x)=a^x\implies f'(x)=a^x\ln{a}，f(x)=e^x\implies f'(x)=e^x
    f(x)=\log_{a}{x}\implies f'(x)={1\over x\ln{a}}，f(x)=\ln{x}\implies f'(x)={1\over x}

    四则运算¶

    [f(x)\pm g(x)]'=f'(x)\pm g'(x)
    [f(x)g(x)]'=f'(x)g(x)+f(x)g'(x)
    [cf(x)]'=cf'(x)
    [{f(x)\over g(x)}]'={f'(x)g(x)-f(x)g'(x)\over [g(x)]^2}\ (\ g(x)\ne 0\ )

    复合函数求导¶

    y=f(g(x))，y=f(u)，u=g(x)
    {y_x}'={y_u}'·{u_x}'

    有多重复合可以一直拆，可以整体使用运算法则。

    (f(g(x)))'=f'(g(x))·g'(x)
    (\ f_1(f_2(f_{...}(f_n(x))))\ )'={f_1}'(f_2(f_{...}(f_n(x))))·{f_2}'(f_{...}(f_n(x)))·{...}\ ·{f_n}'(x)

    运算加速¶

    (e^xf(x))'=e^x(f(x)+f'(x))
    ({f(x)\over e^x})'={f'(x)-f(x)\over e^x}

    [f(x)·g(x)]' 计算：在算式下写出各自的导函数，交叉相乘再相加。

    ({f(x)\over g(x)})' 计算：先写分母 g^2(x) ，再在算式右写出各自的导函数，交叉相乘，斜上线减斜下线作为分子。

    几何意义¶

    f'(x_0) 为 (x_0,f(x_0)) 点的切线斜率

    f(x) 在点 (x_0,f(x_0)) 处的切线方程：

    y-f(x_0)=f'(x_0)(x-x_0)

    公切线：分别设点求切线方程，两条切线 y=kx+b 中的 k 相等， b 相等，从而列出方程组求解切点坐标。

    单调性、极值、最值、凸凹¶

    单调性¶

    f'(x)>0\iff f(x)\nearrow
    f'(x)<0\iff f(x)\searrow

    若 f(x) 在 (a,b) 的任何子区间内非常数函数，则：

    f(x)在(a,b)\nearrow\iff f'(x)\ge 0
    f(x)在(a,b)\searrow\iff f'(x)\le 0

    求单调性：

    1. 求定义域，求导，分式就通分
    2. 有参数时讨论情况
    3. 解不等式 f'(x)>0 或 f'(x)<0

    单调性题型：

    1. 求单调区间、极值
    2. 已知单调性，求参数范围
    3. 存在单调区间，求参数范围

    极值¶

    当 f'(x)=0 时

    f'(x_0-Δx)>0,f'(x_0+Δx)<0\iff f(x_0)为极大值
    f'(x_0-Δx)<0,f'(x_0+Δx)>0\iff f(x_0)为极小值

    求极值：

    1. 求定义域，求导，分式就通分
    2. 求单调性
    3. 判断点：导数为0时、原函数无意义时（好像已经被定义域筛走了吧）、导函数无意义时（如 x=1 ）

    最值¶

    极值与端点处函数值比较即可，注意闭区间和开区间的区别。

    凸凹¶

    f''(x) 为二阶导数。原函数有可能递增也有可能递减。

    凸\iff f'(x)\searrow\iff f''(x)\le 0
    凹\iff f'(x)\nearrow\iff f''(x)\ge 0

    常见函数¶

    （待填坑

    f(x)=x·e^x,f'(x)=(x+1)e^x
    f(x)={e^x\over x},f'(x)={x-1\over x^2}e^x
    f(x)=x·\ln{x},f'(x)=1+\ln{x}
    f(x)={\ln{x}\over x},f'(x)={1-\ln{x}\over x^2}

    三次函数¶

    f(x)=ax^3+bx^2+cx+d\ (a≠0)
    f'(x)=3ax^2+2bx+c
    Δ_3=b^2-3ac

    中心点：（拐点）：（ f''(x)=0 ）

    (-{b\over 3a},f(-{b\over 3a}))

    总体方向：a>0 时总体 \nearrow；a<0 时总体 \searrow 。

    单调性：

    Δ_3>0\implies b^2-3ac>0\implies 图像为 N 型或 И 型

    Δ_3\le 0\implies b^2-3ac\le 0\implies 图像为递增型或递减型

    特别地，Δ_3=0 时，中心点处导数为0。（ f'(-{b\over 3a})=0 ）

    极值点：

    Δ_3>0\implies b^2-3ac>0 时有极值点 x_1,x_2 （关于中心点对称）。

    主要使用二次函数的方法处理 f'(x) 。

    x_{12}={-b\pm\sqrt{Δ_3}\over 3a},x_1+x_2=-{2b\over 3a},x_1x_2={c\over 3a}

    杂项¶

    不知道有什么用

    \lim_{n→\infty}{(1+{1\over n})^n}=e≈2.71828
    \lim_{n→\infty}\ln{(1+{1\over n})^n=1}\implies \lim_{n→\infty}{\ln(1+{1\over n})\over{1\over n}}=1

    令x={1\over n}，则

    \lim_{x→0}{\ln{(1+x)}\over x}=1

    设 P(x,\ln{(1+x)})， \ln(1+x) 过原点

    {\ln{(1+x)}\over x}=k_{OP}\implies \lim_{x→0}{\ln{(1+x)}\over x}=k_{O切}=1

    所以 O 点切线为 y=x，所以 x\ge\ln{(1+x)} （x=0时取等号）

    不知道有什么用

    \lim_{x\to 0}{\sin{x}\over x}=1

    证明：在单位圆上作射线 OA 和 OB，使 \angle{AOx}=\angle{BOx}=\alpha\ (\alpha\in(0,{\pi\over 2}))

    |AB|=2\sin{\alpha}<弧{AB}=2\alpha\ (\alpha={l\over r}) \implies {\sin{\alpha}\over\alpha}<1
    \alpha\to0,弧{AB}\to|AB|\implies \lim_{\alpha\to0}{\sin{\alpha}\over\alpha}=1

    设 P(x,\sin{x}) ，k_{OP}={\sin{x}\over x} ，所以

    \lim_{x\to0}{\sin{x}\over x}=\lim_{x\to0}{k_{O切}}=1

    以曲带直思想

    割圆术，半径为 R 的圆的内接正 n 边形面积为 S_n ，求证：

    \lim_{n\to\infty}{S_n}=\pi R^2

    证明：

    S_n={n\over 2}R^2\sin{2\pi\over n}
    \lim_{n\to\infty}{S_n}=R^2\lim_{n\to\infty}{({n\over 2}\sin{2\pi\over n})}=R^2\lim_{n\to\infty}{\sin{2\pi\over n}\over {2\pi\over n}}·\pi=\pi R^2

    求导 y=x^x 。

    解：

    两边同取对数：

    \ln{y}=x\ln{x}

    两边同求导，自变量为 x ：（这步好像有问题吧）

    \ln{y}·{y_x}'=\ln{x}+1
    {y_x}'={\ln{x}+1\over x\ln{x}}

    y=x-1 是 y=\ln{x} 在 (1,0) 上的切线，证明略
