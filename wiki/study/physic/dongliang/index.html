    动量¶

    • 动量：一个运动的物体所具有的潜在破坏量，矢量。p={m\over v}
    • 冲量：冲击过程中表现出来的破坏力，矢量。I=Ft
    • 碰撞过程中机械能守恒为弹性碰撞，不守恒为非弹性碰撞
    • 完全弹性碰撞：动量、机械能守恒；完全非弹性碰撞：动量、机械能不守恒，动撞静最后共速

    动量定理¶

    • ∑F·t=m·Δv，本质为牛顿第二定律（∑F=m·{Δv\over t}），研究初末状态
    • 动能定理：∑F·x=ΔE_k，区别：动量定理为力对时间累计（冲量），动能定理为力对位移累积（功）

    动量守恒定律¶

    • Δp=m·Δv=0\iff系统合外力为 0
    • 一般形式：m_1\overrightarrow{v_1}+m_2\overrightarrow{v_2}=m_1\overrightarrow{v'_1}+m_2\overrightarrow{v'_2}
    • 本质是牛顿第一定律、动量定理的特殊形式，记得注意方向

    系统合外力为 0 常见形式¶

    只有内力（类碰撞、类爆炸）¶

    • 子弹打木块（地面光滑等）
    • 人船模型（船与水面的阻力忽略不计，人从船的一端走到另一端）
    m·v_人+M·v_船=0，{v_船\over v_人}={m\over M}，x_人+x_船=L_船
    x_人={M\over M+m}L_船，x_船={m\over M+m}L_船
    • 地面光滑的板块模型
    • 弹簧拉缩
    • 带电体排斥或吸引（类似弹簧拉缩）
    • 物块m冲上弧面（斜面）M（水平方向无外力，即 m·Δv_x=0）
      • 三个时刻动量守恒：上去前、上去时（最高点）、上去后（脱离）
    • 桌上小车（金属杆上的环）M用绳子挂了个小球m，给小球m一个初速度v。最高点动量守恒

    内力远大于外力¶

    碰撞（正碰、撞击）、爆炸 瞬间 ，\lim_{Δt→0}

    外力抵消¶

    • 地面不光滑但有外力抵消的板块模型
    • 电磁感应 L-L 模型

    题目¶

    （1）随便算算

    F_A=E_0Q
    F_A=ma\implies a={E_0Q\over m}
    d={1\over 2}at^2\implies t=\sqrt{2d\over a} \implies t=\sqrt{2md\over E_0Q}

    （2）思路：初始状态为距离无穷大，电势能为 0，两物体运动至共速时（即距离最小时）E_{p_m}取得最大值（完全非弹性碰撞），等于系统动能在该过程中的损失量。其中A的初始动能为加速电场对A做的功，可简化计算。

    {v_a}^2=2ad={2E_0Qd \over m}\implies v_a=\sqrt{2E_0Qd \over m}
    v_b=\sqrt{2E_0q_bd \over m_b}=\sqrt{32E_0Qd\over 9m}={4\over 3}\sqrt{2E_0Qd \over m}={4\over 3}v_a
    m_av_a+m_bv_b=(m_a+m_b)v'
    v'={16\over 15}v_a={16\over 15}\sqrt{2E_0Qd \over m}
    E_{p_m}={1\over 2}{m_av_a}^2+{1\over 2}{m_bv_b}^2-{1\over 2}(m_a+m_b){v'}^2 ={1\over2}m{v_a}^2·{1\over 45}
    E_{k_a}={1\over2}m{v_a}^2=E_0Qd\implies E_{p_m}={1\over 45}E_{k_a}={1\over 45}E_0Qd

    （3）思路：B 从头到尾都不改变运动方向，开始 B 离 A 无穷远且电势能为 0，临界情况为最终速度恰好减为 0 且离 A 无穷远，动量守恒且动能守恒，为完全弹性碰撞，公式求解即可。

    {v_b}'={(m_b-m_a)·v_b+2m_av_a\over m_b+m_a}\ge 0
    当v_b=0\ 时，(m_b-m_a)·v_b+2m_av_a=0
    v_b={8\over 3}v_a={8\over 3}\sqrt{2E_0Qd \over m}，v_b=\sqrt{2E_0q_bd \over m_b}=\sqrt{8E_0q_bd \over m}
    解得\ q_b={16\over 9}Q
